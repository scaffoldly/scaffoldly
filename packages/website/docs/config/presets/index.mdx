# Presets

Scaffoldly has been designed with flexibility in mind, allowing you to easily deploy various types of applications using predefined configurations known as presets. These presets simplify the deployment process by providing a set of default settings tailored for specific frameworks or use cases.

The following presets are currently available:

- [**Next.js**](/docs/config/presets/nextjs): A preset for deploying NextJS applications to AWS Lambda.
